📱 iOS (Shadowrocket / Quantumult Lite / Kitsunebi)
نصب

از App Store:

Shadowrocket (پولی)

Quantumult Lite یا Kitsunebi (رایگان)

وارد کردن کانفیگ

Shadowrocket:

پایین صفحه → تب Servers → ضربه روی ＋ →

Type: Vmess/Vless → لینک را پیست کنید

یا از بالا Import from QR Code

Quantumult Lite / Kitsunebi:

تب Settings → Subscription → ضربه روی ＋ → لینک را پیست → Done

اعطای مجوز VPN

در اولین اتصال، iOS از شما اجازهٔ ایجاد پروفایل VPN می‌خواهد؛ حتماً Allow کنید.

در Settings → General → VPN کنترل کنید که پروفایل Connected باشد.

فعال‌سازی DNS امن

Shadowrocket → Settings → Remote DNS → Enable

Quantumult Lite → Settings → DNS → Use Remote DNS

تست اتصال

Safari → ifconfig.me یا whatismyip.com

عیب‌یابی

پس از تغییر کانفیگ: اپ را Force Quit و مجدداً باز کنید.

در صورت عدم اتصال: VPN را Off/On کنید یا دستگاه را ری‌استارت نمایید.

نکات امنیتی و عمومی
همیشه کانفیگ را فقط از منبع رسمی (فروشنده یا Subscription HTTPS) دریافت کنید.

از ویرایشگر مطمئن (VSCode) برای مشاهده/ویرایش JSON استفاده کنید؛ هیچ کاراکتر اضافی یا کامنتی مجاز نیست.

پس از هر به‌روزرسانی کانفیگ، سرویس یا اپ را Restart کنید.

به‌منظور جلوگیری از DNS Leak، همواره گزینه‌های Use Remote DNS یا معادل آن را فعال نگه دارید.

برای پروتکل‌های پیشرفته (VMess/VLess/Trojan) مستندات رسمی V2Ray و کلاینت مربوطه را مرور کنید.
