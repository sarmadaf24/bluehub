📱 iOS (Shadowrocket / Quantumult Lite / Kitsunebi)
ایپ انسٹال کریں

Shadowrocket (ادائیگی والا) یا Quantumult Lite / Kitsunebi (مفت) App Store سے انسٹال کریں۔

کانفیگ درآمد کریں

Shadowrocket:

Servers ٹیب → ＋ → Type: Vmess/Vless → URL پیسٹ کریں یا Import from QR Code۔

Quantumult Lite / Kitsunebi:

Settings → Subscription → ＋ → URL پیسٹ کریں → Done۔

VPN اجازت دیں

پہلی کنکشن پر VPN پروفائل بنانے کی اجازت دیں۔

Settings → General → VPN میں Connected اسٹیٹس دیکھیں۔

محفوظ DNS فعال کریں

Shadowrocket: Settings → Remote DNS → Enable

Quantumult Lite: Settings → DNS → Use Remote DNS

کنکشن ٹیسٹ کریں

Safari میں ifconfig.me یا whatismyip.com کھولیں۔

مسائل حل

نیا کنفیگ لگانے کے بعد ایپ کو Force Quit کر کے دوبارہ کھولیں۔

اگر نا‍صلب ہو تو VPN کو بند/چالو کریں یا ڈیوائس ری اسٹارٹ کریں۔

🔒 بہترین عملی اقدامات
کنفیگ صرف سرکاری اور محفوظ (HTTPS) ذرائع سے حاصل کریں۔

JSON فائل کو معتبر ایڈیٹر (مثلاً VSCode) میں کھولیں—اضافی کومہ یا تبصرے نہ ہوں۔

ہر تبدیلی کے بعد کلائنٹ/سروس کو ری اسٹارٹ کریں۔

DNS لیک روکنے کے لیے Use Remote DNS ہمیشہ آن رکھیں۔

ترقی یافتہ پروٹوکولز (VMess, VLess, Trojan) کے لیے V2Ray کی آفیشل ڈاکیومنٹ ملاحظہ کریں۔
