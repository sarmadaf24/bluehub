🍎 میک‌او‌ایس (V2RayX)
شرائط

macOS 10.12 یا بعد کا ورژن

Catalina+ پر ممکن ہے System Preferences → Security & Privacy میں اجازت درکار ہو

تنصیب

GitHub Releases سے .dmg فائل ڈاؤن لوڈ کریں اور Applications فولڈر میں ڈریگ کریں۔

کانفیگ لوڈ کریں

V2RayX کھولیں۔

مینو میں سے File → Import Subscription (URL کے لیے) یا Import JSON (فائل کے لیے) منتخب کریں۔

سروس شروع کریں

مینو بار (اوپر) میں V2RayX آئیکن پر کلک کریں → Start۔

اگر kernel extension کی اجازت مانگے تو Security & Privacy میں Allow پر کلک کریں۔

پراکسی موڈ منتخب کریں

Global (پورے سسٹم) یا PAC (سمارٹ موڈ)۔

بہتر کارکردگی کے لیے “Bypass LAN & China” بھی آن کیا جا سکتا ہے۔

کنکشن ٹیسٹ کریں

curl ifconfig.me

یا براؤزر سے IP تصدیق کریں۔

اضافی نوٹس

کانفیگ یا سبسکرپشن تبدیل ہونے پر Restart آپشن سے دوبارہ شروع کریں۔

Preferences → DNS میں “Enable Remote DNS” تمام وقت فعال رکھیں۔

🔒 بہترین عملی اقدامات

کنفیگ صرف سرکاری اور محفوظ (HTTPS) ذرائع سے حاصل کریں۔

JSON فائل کو معتبر ایڈیٹر (مثلاً VSCode) میں کھولیں—اضافی کومہ یا تبصرے نہ ہوں۔

ہر تبدیلی کے بعد کلائنٹ/سروس کو ری اسٹارٹ کریں۔

DNS لیک روکنے کے لیے Use Remote DNS ہمیشہ آن رکھیں۔

ترقی یافتہ پروٹوکولز (VMess, VLess, Trojan) کے لیے V2Ray کی آفیشل ڈاکیومنٹ ملاحظہ کریں۔
